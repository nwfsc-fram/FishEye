bs_carousel_image {bsplus}R Documentation

Carousel image

Description

Helper function to generate HTML for a carousel image.

Usage

bs_carousel_image(...)

Arguments

...

additional arguments passed to htmltools::tag$img, typically includes src

Details

This function wraps htmltools::tag$img, but adding a class to center the image in the carousel.

Value

htmltools::tag, </img>
