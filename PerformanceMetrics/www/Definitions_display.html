FISHeries Economics Explorer (FISHEyE)

 

Display

Select multiple metrics:FISHEyE lets you choose whether you would like to create multiple figures, one for each selected vessel/processor category or one for each selected metric. The default is to allow the user to choose multiple categories and one metric, if you would like to look at multiple metrics for one category, you can turn this option on with the “Select multiple metrics” slider:

Show variance:This functionality allows user to include a variance band on the plots. When plotting mean the variance band represents the mean plus or minus the variance. When plotting median the variance band represents the 25th and 75th quartiles.
